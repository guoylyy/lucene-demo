# lucene-demo

本项目为 Lucene 的学习文档，其中包含了几个演示项目。

Lucene 是一个用Java 编写的极其丰富和强大的全文搜索库。您可以使用 Lucene 为数据库对象和各种格式的文档（Microsoft Office 文档、PDF、HTML、文本等）提供全文索引。
